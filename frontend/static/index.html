Quickstart — Upload & Download

                    

                        

                            
Upload a file

                            

                                # cURL

                                curl -X POST /api/v1/upload \

                                  -F "file=@yourfile.pdf"
                            

                        

                        

                            
List files

                            

                                curl /api/v1/files?limit=20&offset=0
                            

                        

                        

                            
Download a file

                            

                                curl -O /api/v1/download/{file_id}
                            

                        

                        

                            
Create a shared link

                            

                                curl -X POST /api/v1/shared-links \

                                  -H "Content-Type: application/json" \

                                  -d '{"file_id":"...","link_type":"public","expires_hours":24}'
                            

                        

                    

                


                
                

                    
Architecture Overview

                    

                        

                            
                            
Erasure Coding

                            
Reed-Solomon 6+3 encoding. Files split into 9 shards — any 6 can reconstruct the original.

                        

                        

                            
                            
Multi-Cloud Burst

                            
Shards distributed across GDrive, OneDrive, Dropbox, S3, and local disk simultaneously.

                        

                        

                            
                            
Auto-Healing

                            
Background workers continuously verify shard integrity and repair lost fragments automatically.
