# Contributing to vit-storage

Thank you for your interest in contributing to the VIT Network ecosystem!

## Workflow

1. Fork the repository and create your branch from `main`.
2. Follow the existing code style and structure.
3. Add or update tests for any behavior change.
4. Ensure CI (lint, build, tests) passes before requesting review.
5. Open a Pull Request using the provided PR template.

## Commit Messages

Use [Conventional Commits](https://www.conventionalcommits.org/) (e.g. `feat:`, `fix:`, `docs:`, `chore:`).

## Code of Conduct

This project adheres to the [VIT Network Code of Conduct](https://github.com/Value-intelligence-trust/.github/blob/main/CODE_OF_CONDUCT.md).

## Reporting Issues

Use the issue templates provided. Include reproduction steps, expected vs actual behavior, and environment details.
